## 現象

昨日久しぶりにブログを投稿しましたが...
今朝確認したところ、投稿されていないではありませんか。。。<br/>
何故だろう、リポジトリへの push がうまくいっていないかな
と思い GitHub を確認したところ、しっかりと push されている。<br/>
次に Gatsby Cloud のサイトを見にいったら、何やら build failed らしき
文言が。。。

しかし、ローカル環境ではしっかりとビルドできているし、Error も 0 件だし
何故だろう。。。

## 対応

最終的に、何かコードが間違っていたとかではありませんでした。
Gatsby Cloud 上で Trigger build というボタンがあるので、そこを
クリックすれば手動で再度 build ができ、問題なく完了しました。

## 結論

エラーとか特にでていないのに Gatsby Cloud への反映がうまくいかなければ
再度 build してみましょう。
