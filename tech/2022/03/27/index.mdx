## 何をしたいのか

VS Code にはプログラミングをしながら、ユーザーが入力した文字に応じて、入力したいワードを予測して
提案してくれる機能がありますね。

非常に便利ではあるのですが、使うつもりのない予測ワードが表示された時に、無視しようとしてもキーボードの
入力を行なったら、勝手に予測ワードが入力されてしまうなんてことも...

最近競技プログラミングに参加しながら、これに非常に困っており、なんとかしようと思い立ち調べてみました。

## どうすれば良いのか

どうやら、この機能については、設定でカスタマイズできるようですね。

公式のドキュメントは[こちら](https://code.visualstudio.com/docs/editor/intellisense)

ということで、settings.json に

```json
"editor.acceptSuggestionOnCommitCharacter": false
```

を指定することで、誤入力がなくなりました。

これで快適な競プロライフを送ることができそうです。
