皆さんこんにちは、こんばんは、初めまして、当サイト管理人の masayamatu です。<br/>
やっとオリジナルサイトの開設に辿り着けたので、当記事でサイト周辺のことについて、
いろいろとまとめていきたいと思います。

## 開設の経緯について

&emsp;当サイト管理人の masayamatu は、日頃 IT エンジニアをしています。
C#、Perl、JavaScript、SQL 等いろいろな、技術を用いて仕事
をしていますが、分からないことだらけで、Google さんがいないと
生きていけないような状況です。<br/>
&emsp;しかし、ある時気づいたのです。私が学んでいる記事も誰かが
書いてくれて、無料で公開していくれているものなのだと...<br/>
&emsp;そうなると、こんな気持ちが芽生えてきました、
「学ばせてもらってばかりでは申し訳ない。自分の知見も公開していき、世の中の役に立とうではないか。」<br/>
&nbsp;こんな経緯で、当サイトが生まれました。さて、綺麗事を書いてしまいましたが、実際、自分で学んだこと
のアウトプットを通して理解を確実にすることや、人に説明する能力を培う、自分で作成した記事を
リファレンスとして用いたい etc...<br/>といった感じで、自分のために作成したという所も大きいです。

## 使用技術について

&emsp;結論を先に書くと<strong>Gatsby.JS</strong>という、React ベースの JavaScript
フレームワークを利用して作成しています。
オリジナルの Web サイトやブログを作成するにしても、すでに
技術系ブログのプラットフォームとしては、qiita や zenn と
言った素晴らしいサイトがすでにありますし、自分でオリジナル
で構築していくとしても、Word Press や Wix といった便利な
ブログ構築ソフトも出ています。<br/>
&emsp;上記のサービスを初めから利用すれば良いものを、
自分で一からオリジナルのサイトを構築していきたいという**謎のレゲエマン精神**が芽生えてしまい、
一からサイトを組み上げることにしました。<br/>
&emsp;しかし、本当に一からブログサイトを構築していくには、莫大な時間がかかってしまいます。
働きながら他にいろいろと勉強したい身としては、そこまでの時間は取れないので、何かいい塩梅な
物はないかと調べていたら、最近は静的サイトジェネレータという物が出てきているというではないですか。
候補はいろいろとあるみたいですね。以下のサイトに静的サイトジェネレータがまとめられているようです。<br/>

[https://jamstack.org/generators/](https://jamstack.org/generators/ "https://jamstack.org/generators/")<br/>

いろいろありすぎて、正直**どれが良いのか分からない**のが本音なのですが、
今回は、以下の理由から Gatsby.JS でサイトを作っていくことにしました。

- React ベースでできており、GraphQL が採用されており、モダンなフロントエンド開発を学べる。

- 人気が高くある程度ドキュメント等は充実しているが、日本ではまだ少数派(少数派になりたいという**謎のレゲエマン精神**)

&emsp;Gatsby.JS であれば、コマンドで一撃でブログサイトの雛形まで作れるようですが、勉強も兼ねて
なので、一から Gatsby.JS の tutorial を読みながら作成していきました。tutorial のリンクを
以下に貼り付けておきます。

[https://www.gatsbyjs.com/docs/tutorial/](https://www.gatsbyjs.com/docs/tutorial/)<br/>

React コンポーネントや、 GraphQL、MDX など日頃の業務では、成熟した技術ばかりを使っている身としては
慣れない概念ばかりで大変でしたが、なんとかここまでは来れました。(最先端のフロントエンド開発が大変だと
言われる所以を身に染みて感じた...)

## 今後の課題

&emsp;恐らく当サイトをご覧になった皆さんの第一印象は寂しいサイトだなという印象かと思います。
まだまだ、tutorial レベルから抜け出していないので、今後の以下のような事を実現していき、もう
少し良い感じのサイトにしていきたいと思っています。

- アイコンの導入により視覚的にリッチなサイトする。
- ヘッダー、フッター、サイドメニュー当の拡充
- ブログ一覧表示のカードコンポーネントデザインの導入
- 記事のカテゴリーの導入と、カテゴリー別一覧画面の導入

まずは、こんなところでしょうか。
デザインについては、Semantic UI とか bootstrap とか、その辺りのデザインフレームワークを導入
していこうと思っています。世界レベルのエンジニアさんになると、Gatsby.JS で以下のような
凄いサイトがバリバリ作れるようです。

[https://www.gatsbyjs.com/showcase/](https://www.gatsbyjs.com/showcase/)

なお、この記事のソースは MDX という形式で書いているのですが
、全然慣れなくて使いこなせていません。まずはここから慣れないと...(実際ブログには
もったいないぐらいいろいろできるようです。以下リンクのような感じ)<br/>

[https://frontarm.com/demoboard/](https://frontarm.com/demoboard/)<br/>

まあ、向こう２、３年インプットよりの学習に時間を使いたいので、サイトの改修はやっこらかもしれません。
ブログの更新は、自分の苦手な頭の使い方をするので、意識的に取り組む時間は取りたいと思ってます。
